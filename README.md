# homebrew-claude-prism

Homebrew tap for [claude-prism](https://github.com/tznthou/claude-prism) — Multi-AI provider toolkit for Claude Code.

## Install

```bash
brew tap tznthou/claude-prism
brew install claud-prism-aireview
```

Then run `claude-prism` to install commands to `~/.claude/`.

## Uninstall

```bash
brew uninstall claud-prism-aireview
brew untap tznthou/claude-prism
```
